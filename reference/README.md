# Reference Directory

This directory stores background material used to interpret the implemented
methods and the figure-analysis scripts.

The files in this directory are **not** runtime dependencies. They are
explanatory material for method alignment and figure interpretation.

## Main Reference File

- [20260305_centroid_manuscript_v4_lpr3.docx](20260305_centroid_manuscript_v4_lpr3.docx)

This manuscript draft is useful for understanding:

- the definition of the Relative Centroid Index (`RCI`)
- the difference between precipitation and runoff centroids
- why centroid position is measured along cumulative main-stem distance
- how Pfafstetter levels are used for hierarchical analyses
- the figure-level analysis logic implemented in `scripts/`

If your goal is to run the code rather than interpret the paper workflow, start
with:

- [README.md](../README.md)

and then use:

- [scripts/README.md](../scripts/README.md)
- [results/README.md](../results/README.md)
